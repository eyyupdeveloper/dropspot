# DropSpot

Projeye Baslama Zamani: 202511061817

DropSpot, sınırlı stokla yayınlanan ürünler için bekleme listesi tabanlı bir platformdur.
Kullanıcılar drop'lara katılabilir, bekleme listesine girebilir ve zamanı geldiğinde hak talep edebilir.

Bu proje, Alpaco Teknoloji Full Stack Developer case çalışması kapsamında geliştirilmiştir.
